# Job Hub Portal




#### run the app loccaly 

```sh
npm install
cd client
npm install
```

#### from root dir
```sh
npm run dev
```


#### production build

```sh
cd client
npm run build
```

then move content of /client/public folder to -> /public


```sh
/client/public  -> /public
```

and then

#### from root dir
```sh
node server.js
```
